# Project Brief

## TWW Companion

**What:** A mobile-first web app that supports people during their Two-Week Wait (TWW) — the 14 days between ovulation and when a pregnancy test is reliable.

**Why:** The TWW is emotionally intense. People symptom-spot obsessively, feel isolated, and lack a dedicated supportive tool. Existing period trackers are clinical and cold.

**Who:** Anyone TTC (trying to conceive) currently in or preparing for their TWW.

**Core Features (Built):**
1. **TWW Timeline** — personalized 14-day DPO countdown based on ovulation date
2. **Daily Body Guide** — what's happening biologically each day
3. **Symptom Logging** — quick chip-based daily tracking with optional notes
4. **"Should I Test?" Indicator** — evidence-based testing guidance with reliability levels
5. **Community Feed** — post, reply, heart, and delete in a supportive space
6. **Encouragement System** — rotating supportive micro-copy

**Design Principles:**
- Warm and emotionally supportive (not clinical)
- Mobile-first with purposeful tablet/desktop layouts
- Glassmorphic aesthetic with frosted cards and gradient backgrounds
- Fraunces serif for personality, Nunito for readability
- Indigo/violet color palette

**Tech:** React 18 + Vite + Tailwind CSS v4, deployed on Vercel
