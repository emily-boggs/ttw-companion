# TWW Companion

A beautiful, modern companion app for the Two Week Wait — the 14-day period between ovulation and when a pregnancy test becomes reliable.

🔗 **[Live Site](https://ttw-companion.vercel.app/)**

## About

TWW Companion helps you navigate each day of the two week wait with science-backed guidance, symptom tracking, and community support. Built with a glassmorphic design system featuring soft gradients, frosted glass cards, and a calming blue-indigo-violet palette.

## Features

- **Daily Guides** — Personalized content for each DPO (days past ovulation) explaining what's happening biologically
- **Symptom Tracking** — Log how you're feeling each day with quick-tap chips and optional notes
- **Timeline View** — See your full 14-day journey at a glance with logged symptoms
- **Test Reliability** — Science-based guidance on when testing is meaningful
- **Community** — Share thoughts, encouragement, and connect with others in the same journey
- **Responsive Design** — Purposeful layouts for mobile, tablet, and desktop with an M3-inspired navigation rail

## Tech Stack

- **React 18** + Vite 5
- **Tailwind CSS v4** with custom theme
- **React Router** for client-side routing
- **Lucide React** for icons
- **Google Fonts** — Fraunces (serif headlines) + Nunito (body)
- **localStorage** for client-side data persistence

## Getting Started

```bash
cd tww-companion
npm install
npm run dev
```
